# OpenClaw 教程

OpenClaw 是一个开源的 AI 编程助手工具。

## 安装

```bash
pip install openclaw
```

## 快速开始

```python
from openclaw import Assistant

assistant = Assistant()
code = assistant.generate("Write a Python function to calculate fibonacci")
print(code)
```
