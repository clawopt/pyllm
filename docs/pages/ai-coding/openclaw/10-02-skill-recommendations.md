# 常用Skill推荐清单

ClawHub上有上万个技能，以下是根据社区反馈精选的推荐清单。

## 核心必备

| 技能名称 | 描述 | 评分 | 推荐理由 |
|---------|------|------|---------|
| `agent-browser` | 浏览器自动化 | ⭐ 4.9 | 官方维护，功能最全 |
| `file_ops` | 文件操作 | ⭐ 4.8 | 内置技能，日常必备 |
| `web_search` | 联网搜索 | ⭐ 4.7 | 信息获取必备 |
| `email` | 邮件发送 | ⭐ 4.6 | 工作沟通必备 |

## 安装推荐组合

**个人开发者：**

```bash
openclaw hub install agent-browser file_ops web_search git_ops summarizer
```

**内容创作者：**

```bash
openclaw hub install agent-browser summarizer translator image_gen blog_publisher
```

**企业用户：**

```bash
openclaw hub install agent-browser file_ops email calendar notion_sync secureclaw
```

**数据分析师：**

```bash
openclaw hub install data_visualizer sql_assistant excel_processor report_generator
```

---

技能安装前，建议使用 `openclaw hub show <skill_name>` 查看详情，确认安全后再安装。
