# OpenCode 教程

OpenCode 是一个轻量级的 AI 代码助手。

## 安装

```bash
pip install opencode
```

## 快速开始

```python
from opencode import CodeAssistant

assistant = CodeAssistant()
result = assistant.generate("Create a REST API with Flask")
print(result.code)
```
