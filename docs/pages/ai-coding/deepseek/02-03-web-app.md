# 2.3 网页版与 App 使用

> **不想写代码？打开浏览器就能用 DeepSeek——免费、免注册、支持深度思考模式。**

---

## 这一节在讲什么？

不是所有场景都需要写代码调 API——有时候你只是想快速问个问题、分析一个文档、或者体验一下 DeepSeek 的推理能力。DeepSeek 提供了免费的网页版和 App，支持 V3.2 和 R1 模型，支持文件上传和联网搜索。这一节我们快速过一遍网页版和 App 的使用方法。

---

## chat.deepseek.com：免费网页版

打开浏览器，访问 [chat.deepseek.com](https://chat.deepseek.com)，你就可以直接跟 DeepSeek 对话了。

### 模型选择

网页版提供两种模型：

- **DeepSeek-V3.2**：默认模型，快速回答，适合日常对话和代码生成
- **DeepSeek-R1（深度思考）**：点击"深度思考"按钮开启，适合复杂推理和数学问题

### 文件上传

网页版支持上传文件，DeepSeek 会读取文件内容并回答你的问题：

- 支持的格式：PDF、Word、Excel、PPT、TXT、图片等
- 上传后可以问："总结这个 PDF 的要点"、"这个 Excel 里的数据有什么规律"
- 文件大小限制：单个文件最大 100MB

### 联网搜索

网页版和 App 端支持联网搜索——DeepSeek 会先搜索互联网上的相关信息，然后基于搜索结果回答你的问题。这在需要最新信息的场景下很有用：

- "2025 年最新的 Python 版本是什么？"
- "React 19 有什么新特性？"
- "今天的天气怎么样？"

### 多轮对话

网页版支持多轮对话——你可以跟 DeepSeek 持续交流，它会记住之前的对话内容。但注意，对话太长时 DeepSeek 可能会"遗忘"早期的内容——这是因为上下文窗口有限。

## DeepSeek App

DeepSeek 提供了 iOS 和 Android App，功能跟网页版类似，但更方便：

- **iOS**：在 App Store 搜索"DeepSeek"
- **Android**：在 Google Play 搜索"DeepSeek"

App 的优势：
- 随时随地使用，不需要打开电脑
- 支持语音输入
- 支持拍照上传
- 支持联网搜索

## 常见误区

**误区一：网页版免费意味着 API 也免费**

不是。网页版是 DeepSeek 提供的免费服务，但有使用频率限制（每小时/每天的请求次数有限）。API 是按量计费的，没有频率限制，适合集成到应用中。

**误区二：网页版的回答质量跟 API 一样**

基本一样，但有一些细微差别：网页版可能有一些额外的安全过滤和内容审核；API 的参数（temperature、max_tokens 等）可以自定义，网页版不能。

**误区三：网页版支持 Function Calling**

不支持。Function Calling 是 API 专属功能，需要通过代码调用。网页版只能进行对话。

**误区四：网页版的数据会用于训练**

DeepSeek 官方声明不会使用 API 数据训练模型。网页版的隐私政策建议查看官方说明——如果你担心数据隐私，建议使用 API 调用或本地部署。
