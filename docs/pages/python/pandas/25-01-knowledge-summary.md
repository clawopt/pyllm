## 核心 API 速查表

### 数据创建

| 操作 | API | 示例 |
|------|-----|------|
| 从列表 | `pd.DataFrame()` | `pd.DataFrame({'a': [1,2,3]})` |
| 从字典列表 | `pd.DataFrame([dicts])` | 自动对齐列 |
| 从 NumPy | `pd.DataFrame(array)` | 指定行列名 |
| 空 DataFrame | `pd.DataFrame()` | 后续添加数据 |
| 时间范围 | `pd.date_range()` | `freq='D'/'h'/'ME'` |
| 分类 | `pd.Categorical()` | `categories=, ordered=` |

### 数据查看

| 操作 | API | 注意事项 |
|------|-----|---------|
| 前 N 行 | `.head(n)` | 默认5行 |
| 尾部 N 行 | `.tail(n)` | |
| 随机 N 行 | `.sample(n)` | 设置 random_state |
| 信息摘要 | `.info(memory_usage=True)` | 含每列内存 |
| 描述统计 | `.describe()` | 数值列统计量 |
| 形状 | `.shape` | `(rows, cols)` |

### 数据筛选

| 场景 | 方法 | 示例 |
|------|------|------|
| 单条件 | `df[col] > val` | `df[df['score'] > 85]` |
| 多条件(AND) | `&` 或 `.loc[]` | `df[(a>0) & (b<5)]` |
| 多条件(OR) | `\|` | `df[(a>0) \| (b>0)]` |
| 文本包含 | `.str.contains()` | `df['text'].str.contains('kw')` |
| 列表选择 | `isin()` | `df[df['m'].isin(['A','B'])` |
| 范围 | `between()` | `df['val'].between(10, 50)` |
| TopN | `.nlargest(n)` | 比 sort+head 快 O(n log k) |
| 自然语言 | `.query()` | `df.query("a > 5 and b < 3")` |

### 分组聚合

| 操作 | API | 返回形状 |
|------|-----|----------|
| 单个统计 | `.agg('mean')` | Series |
| 多统计 | `.agg(['mean','count'])` | DataFrame |
| 多列多统计 | `.groupby().agg({...})` | MultiIndex |
| 自定义 | `.agg(lambda x: ...)` | 任意 |
| 命名聚合 | `.agg(name='mean')` | 扁展列名 |
| 转换(同形) | `.transform('mean')` | 与原等长 |
| 过滤组 | `.filter(lambda g: len(g)>5)` | 减少组数 |

### 排序

| 操作 | API | 参数 |
|------|-----|-------|
| 值排序 | `.sort_values(col)` | ascending= |
| 多键排序 | `.sort_values([c1,c2])` | 先 c1 后 c2 |
| 排名 | `.rank(method='dense')` | dense/min/max/first |
| 百分位 | `.rank(pct=True)` | 0-1 范围 |
| 变化率 | `.pct_change()` | period=N |
| TopN | `.nlargest(n, col)` | 最快方式 |

### 合并连接

| 类型 | API | 条件 |
|------|-----|------|
| 内连接 | `merge(a,b, on='key', how='inner')` | 两边都有 |
| 左连接 | `..., how='left'` | 左表全保留 |
| 外连接 | `..., how='outer'` | 并集 |
| 索引连接 | `a.join(b)` | 按索引 |
| 纵向堆叠 | `concat([df1,df2])` | 行方向追加 |
| 横向拼接 | `concat([df1,df2], axis=1)` | 列方向追加 |

### 时间序列

| 操作 | API | 说明 |
|------|-----|------|
| 解析时间 | `pd.to_datetime(s, format=)` | 加速 |
| 创建序列 | `pd.date_range(start, freq='D')` | D/h/MS |
| 重采样降频 | `.resample('D').mean()` | 聚合 |
| 滚动窗口 | `.rolling(window).mean()` | 移动平均 |
| 扩展窗口 | `.expanding().sum()` | 累计 |
| 偏移 | `.shift(1)` | 行偏移 |
| 差分 | `.diff()` | 一阶/多阶 |
| 变化率 | `.pct_change()` | 增长% |

## 性能优化速查

| 问题 | 解决方案 | 效果 |
|------|---------|------|
| 循环慢 | 用向量化 / itertuples | **10-100x** |
| 字符串内存大 | 转 category / string[pyarrow] | **60-95%** |
| int64 太大 | downcast 到 int32/int16 | **50-75%** |
| float64 不需精度 | downcast to float32 | **50%** |
| 大文件读取慢 | dtype + usecols + chunksize | **3-10x** |
| CSV 大而慢 | 改用 Parquet | **6.6x 读写** |
| groupby 慢 | 键列转 category | **5-10x** |
| apply 必须用 | 检查能否向量化替代 | — |

## LLM 场景速查

| 任务 | 关键操作 | 典型代码模式 |
|------|---------|-------------|
| SFT 数据清洗 | filter + quality score | `mask = (len>5) & (len>15)` |
| SFT 分层抽样 | groupby + sample | `stratified_sample(df, 'tier', weights)` |
| 模型评估矩阵 | pivot_table | `pivot_table(index='model', columns='task', values='score')` |
| API 成本计算 | merge + price table | `merge(logs, prices, on='model')` |
| RAG 检索 | cosine similarity | `@vectors @ query_vec` |
| 训练 Loss 分析 | rolling + diff | `loss.rolling(20).mean()` |
| 日志分析 | resample('h').count() | `resample('h').agg({'calls':'sum'})` |
